## Как получить

**1. Поставьте звезду репозиторию.** &nbsp;Это вся цена.

**2. Войдите через GitHub** &nbsp;на [ownowl.app](https://ownowl.app).

**3. Выберите имя.** &nbsp;Тип записи A или CNAME, значение, готово. Работает через секунды.

<br>

## Правила

**Один на аккаунт.** &nbsp;Навсегда. Имена не выдаются повторно.

**Живёт, пока стоит звезда.** &nbsp;Проверка ежедневно. Снимете звезду, запись удалится.

**Без изменений.** &nbsp;Чтобы освободить имя, снимите звезду.

**От 3 до 63 символов.** &nbsp;Строчные латинские буквы, цифры и дефисы.

**Цель CNAME должна резолвиться.** &nbsp;Ссылка на несуществующий хост открывает путь к захвату поддомена.

**Только публичный IPv4.** &nbsp;Частные, петлевые и link-local диапазоны запрещены.

**Ничего вредоносного.** &nbsp;Никакого фишинга, вредоносного ПО и противоправного содержимого.
