## 如何领取

**1. 给本仓库点个 star。** &nbsp;这就是全部代价。

**2. 用 GitHub 登录** &nbsp;[ownowl.app](https://ownowl.app)。

**3. 选一个名字。** &nbsp;选择 A 或 CNAME，填写记录值，完成。几秒内生效。

<br>

## 规则

**每账号限一个。** &nbsp;永久有效，名字不会被回收再分配。

**与你的 star 共存亡。** &nbsp;每天复查，取消 star 后记录会被删除。

**不可修改。** &nbsp;若要释放名字，请取消 star。

**3 到 63 个字符。** &nbsp;仅限小写字母、数字和连字符。

**CNAME 目标必须可解析。** &nbsp;指向无人拥有的主机，等于给子域名劫持留门。

**仅限公网 IPv4。** &nbsp;不接受私有、回环或链路本地地址段。

**禁止有害内容。** &nbsp;不得用于钓鱼、恶意软件或任何违法用途。
